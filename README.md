# MineSweep

Command Line version of MineSweeper for Unix-like systems (GNU/Linux, macOS, BSD).

<img src="assets/minesweep.png"></img>

## Prerequisites

- C++11 compiler

## Installation

```bash
$ git clone https://github.com/eliot-akira/minesweep.git
$ cd minesweep
$ make
$ ./mine
```
